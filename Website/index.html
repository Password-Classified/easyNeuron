easyNeuron

The easiest way to create neural networks for all of your data science projects, robotics and more!

easyNeuron Guides User Docs Developer Docs

easyNeuron is the easiest new way to create neural networks from scratch in Python. With built in datasets, preprocessing functions and neural network tools to build and create models, all is packaged within a small, lightweight module to simplify your workflow and give quick results and responses.

Written in raw Python, easyNeuron can not only run on a computer, but on other devices that don't support pip install module commands.
